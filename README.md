# README

This notebook processes the 'Skycity - The City Landscape Dataset' from Kaggle. It performs the following steps:

1.  **Dataset Download and Extraction:** Downloads the dataset from Kaggle and extracts its contents.
2.  **Data Loading and Preparation:** Defines functions to load class names and prepare image data generators for training and validation.
3.  **Data Visualization:** Visualizes a random selection of images with their corresponding labels.
4.  **Model Building:** Constructs a Convolutional Neural Network (CNN) model.
5.  **Model Training:** Trains the CNN model with early stopping and saves the best model based on validation accuracy.
6.  **Results Visualization:** Plots the training and validation accuracy and loss over epochs.
7.  **Prediction and Evaluation:** Predicts the class of a random image from the dataset and displays the results, along with the best validation metrics.

**Dataset Source:**
[Skycity - The City Landscape Dataset on Kaggle](https://www.kaggle.com/datasets/yessicatuteja/skycity-the-city-landscape-dataset)
